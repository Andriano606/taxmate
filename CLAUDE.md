# TaxMate — контроль податків ФОП через API ДПС (Rails-застосунок)

> Точка входу для будь-якого Claude-агента, що працює в цьому проєкті. Прочитай повністю,
> деталі — у `docs/`. Тут два виміри: **предметний** (податки ФОП) і **технічний** (Rails-стек).
> Як підняти/запустити застосунок — у `README.md`. Це — про «що/навіщо», домен і правила роботи.

## Що це за проєкт

**TaxMate** — персональний інструмент власника (ФОП 3 група, IT, єдиний податок 5%), щоб
**самостійно вести податки без бухгалтера**: перевіряти борги/штрафи/пеню, бачити, чи всі декларації
подані й вчасно, тягнути дохід із банку, і (в перспективі) **подавати декларації та платити** —
усе через **приватний API Електронного кабінету ДПС** (`cabinet.tax.gov.ua`) та **monobank Personal API**.

Технічно це **Rails 8.1 застосунок** (Ruby 3.4.8, Node 22, Yarn, Vite/Vue/Turbo/Stimulus, Solid Queue,
Active Storage→MinIO, Kamal). Стек і запуск — у `README.md`.

Ключова ідея безпеки: **КЕП-ключ і токени ніколи не покидають машину власника**; підпис ДСТУ 4145
накладається локально, далі — звичайні HTTPS-запити.

## Як запустити перевірку (Ruby-клієнти)

```ruby
# з кореня Rails-проєкту:  bin/rails runner "..."
signer = Dps::KepSigner.new(
  key_path: "../pb_3525511937.jks", cert_path: "../sign.crt", password_path: "../password.txt")
dps = Dps::Client.new(rnokpp: "3525511937", signer: signer)
dps.debt                       # борг; [] = боргу немає
dps.settlements(year: 2026)    # стан розрахунків
dps.declarations(year: 2026)   # подані декларації

mono = Monobank::Client.new(token: File.read("../mono-token.txt").strip)
mono.fop_accounts              # ФОП-рахунки
mono.credits(account: id, from: Time.zone.local(2026,4,1), to: Time.zone.local(2026,7,1))
```

Тести: `bundle exec rspec spec/clients`. Запуск застосунку: `bin/dev` (див. `README.md`).

## Правила роботи для агента (важливо)

1. **Секрети.** Ніколи не друкуй у чат вміст `password.txt` / `mono-token.txt` / приватного ключа.
   Пароль і токен читай із файлу, не проси текстом у переписці.
2. **Мова.** Власник спілкується українською і не є технічним фахівцем у податках — пояснюй простими
   словами, давай практичні кроки.
3. **Обережність із діями назовні.** GET-запити (перевірка) — безпечні. Будь-яка **подача** документа
   чи **оплата** — незворотні дії з юридичними наслідками; спершу підтверджуй із власником.
4. **Ліміт API.** ДПС ~1000/добу на токен; monobank — виписка 1 запит/60с, діапазон ≤31 день.
5. **Актуальність.** Ставки/ліміти/строки та IBAN бюджетних рахунків змінюються щороку — не бери числа
   з голови, тягни з `/ta/splatp` та звіряй із `docs/02-tax-rules-fop3.md`.
6. **Достовірність.** Якщо чогось не перевірив у даних ДПС/банку — так і кажи, не вигадуй суми.
